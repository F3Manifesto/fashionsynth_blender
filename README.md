Fashion Synth Blender Plugin

***still in dev***

Plugin:
- Download zip
- Edit > Preferences > Add-ons
- Upload zip and enable
- Fashion Synth appears in Sidebar

Script:
- Blender script mode
- Past script_complete.py
- Run
- Fashion Synth appears in Sidebar
